# Reaper
A python conversion on the Reaper multi-tool

# Help

This tool was made by KrisIsHere to help you manage all of your hacking tools in one place.
Reaper's purpose is to make your life easier by compiling all of your tools in the same place so you dont have to look for them all over your computer.

# Help 

How to use?

1.Open the tool

2.Type the name of your linux account (eg. kris@skid kris is the name)

3.type your password (default is root)

4.choose your tool

5.enjoy

# Can you autologin?

To autologin type "autologin" in the terminal once you log in
To disable autologin type "stop_autologin"
